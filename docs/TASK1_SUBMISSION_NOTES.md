# Task-1 Submission Notes

## Final Project Title
Personalized RAG-Based AI Tutoring Platform with Quiz Generation and Weakness Analytics

## Short Registration-Sheet Description
A learner-centered educational AI platform in which students upload PDFs, ask document-grounded questions, take AI-generated quizzes, and receive weakness-focused analytics for personalized revision support.

## One-Paragraph Abstract
This project presents a personalized AI tutoring platform for educational PDFs using Retrieval-Augmented Generation (RAG), quiz generation, grading, and learner analytics. Students upload study documents, interact with them through grounded chat, and take quizzes generated from the same material. Unlike basic PDF chatbots or one-shot quiz generators, the proposed system also analyzes quiz performance to identify weak concepts and visualize progress through an analytics dashboard. The current implementation already provides the core end-to-end workflow, including document upload, indexing, chat, quiz generation, grading, weakness extraction, and analytics. The next stage of the project will strengthen personalization by using stored learner weaknesses to influence future quizzes and recommendations.

## Differentiation Paragraph
This project should not be presented as a plain PDF-to-quiz generator. Its main distinction is that it combines document-grounded tutoring with learner-performance analysis. A simpler system may only generate questions from uploaded material, while this platform also grades responses, identifies weak concepts, and maintains user-specific analytics to support targeted revision. In that sense, the project is learner-centric rather than only document-centric. It is also not being claimed as a multi-agent system, because the current architecture is better described as a personalized RAG tutoring platform with an adaptive-learning direction.
