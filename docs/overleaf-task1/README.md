# Overleaf Files

This folder contains an Overleaf-ready Task-1 proposal package.

## Files
- `main.tex` - main LaTeX proposal file
- `references.bib` - bibliography file for citations

## How to use in Overleaf
1. Create a new blank project in Overleaf.
2. Upload both files from this folder.
3. Set `main.tex` as the main file if Overleaf does not auto-detect it.
4. Replace the placeholder author names in `main.tex`.
5. Compile with PDFLaTeX + BibTeX.

## Notes
- The wording keeps the proposal focused on personalized RAG tutoring, quiz generation, and weakness analytics.
- The proposal does not explicitly position the project as a multi-agent system.
